# CM2604 Machine Learning Coursework

## Overview
Binary classification for Telco Customer Churn using Decision Tree and Neural Network.

## Files
- churn_analysis.ipynb: Main notebook
- WA_Fn-UseC_-Telco-Customer-Churn.xlsx: Dataset
- requirements.txt: Dependencies

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the notebook.
